# Customer Churn Prediction

## 📌 Project Overview
This project predicts whether a customer will leave a service based on historical data.

## 🚀 Features
- Data preprocessing
- Label encoding
- Feature scaling
- Logistic Regression model
- SMOTE for handling imbalance

## 📊 Dataset
- Telco Customer Churn dataset (Kaggle)

## 🧠 Model Used
- Logistic Regression

## 📈 Output
- Churn prediction (Yes/No)
- Accuracy score
- Confusion matrix

## ▶️ How to Run
1. Open notebook in Google Colab
2. Upload dataset
3. Run all cells
